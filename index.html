Como publicar no GitHub

  1. Clique em "Salvar" — um arquivo JSON será baixado
  2. Abra o arquivo e copie todo o conteúdo
  3. Acesse github.com/fernandoneto1210-sys/orcamentos-oficina
  4. Clique em data → orcamentos.json
  5. Clique no lápis ✏️ para editar
  6. Selecione tudo (Ctrl+A) e cole o novo conteúdo
  7. Clique em "Commit changes"
  8. Aguarde 2-3 minutos e teste o link! ✅

Configuração JSONBin.io (Salvar online)

Com o JSONBin configurado, ao clicar em "Salvar" o orçamento é publicado instantaneamente — sem precisar mexer no GitHub!

Como criar: Acesse jsonbin.io → Sign Up → Create Bin → cole o JSON atual → copie o Bin ID e a API Key abaixo.

Sem JSONBin? Use o método manual

  1. Clique em "Salvar" no painel — baixa o arquivo JSON
  2. Acesse o GitHub e edite data/orcamentos.json
  3. Substitua o conteúdo e faça o commit
  4. Aguarde 2-3 minutos para o link atualizar
